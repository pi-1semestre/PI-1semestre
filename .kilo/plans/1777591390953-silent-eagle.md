# Technical Plan for Voluntario and EmailVoluntario Implementation

## Overview
Implement the Voluntario and EmailVoluntario entities, repositories, services, and controllers following the existing architectural patterns in the codebase (Spring Boot with JPA/Hibernate).

## Mapping Strategy (JPA/Hibernate)
- Use Jakarta Persistence API annotations for entity mapping.
- Entities will follow the established pattern: @Entity, @Table(name="tb_..."), @Id with GenerationType.IDENTITY, @Column with appropriate constraints.
- Relationships mapped using @OneToOne for unidirectional association.

## Relationship Type
- One-to-One unidirectional relationship from Voluntario to EmailVoluntario.
- Owning side: Voluntario (FK resides in tb_voluntarios table).
- EmailVoluntario entity has no reference back to Voluntario.

## Implementation Logic by Layer

### Entities
- **EmailVoluntario**: Simple entity with idEmailVoluntario (Long, PK, AI) and email (String, VARCHAR 255).
- **Voluntario**: Entity with idVoluntario (Long, PK, AI), nome (String, VARCHAR 45), idade (Integer, INT), and @OneToOne EmailVoluntario with @JoinColumn(name="tb_emailVoluntario_idEmailVoluntario").
- Constructors: Default and parameterized.
- Getters and setters for all fields.

### Repositories
- **EmailVoluntarioRepository**: Interface extending JpaRepository<EmailVoluntario, Long>, @Repository annotation.
- **VoluntarioRepository**: Interface extending JpaRepository<Voluntario, Long>, @Repository annotation.
- No custom queries needed initially.

### Services
- **EmailVoluntarioService**: @Service class with @Autowired repository, providing findAll(), findById(Long), save(EmailVoluntario), deleteById(Long).
- **VoluntarioService**: @Service class with @Autowired repository, providing findAll(), findById(Long), save(Voluntario), deleteById(Long).
- Business logic: Standard CRUD operations as per existing services.

### Controllers
- **EmailVoluntarioController**: @RestController with @RequestMapping("/api/v1/emailVoluntarios"), @Autowired service.
  - GET / : List all
  - GET /{id} : Get by id
  - POST / : Create
  - PUT /{id} : Update
  - DELETE /{id} : Delete
- **VoluntarioController**: @RestController with @RequestMapping("/api/v1/voluntarios"), @Autowired service.
  - GET / : List all
  - GET /{id} : Get by id
  - POST / : Create

- Responses: Use ResponseEntity with appropriate HTTP status codes.

import jakarta.persistence.*;

@Entity

        this.idEmailVoluntario = idEmailVoluntario;
        this.email = email;

        return idEmailVoluntario;
    }

    public void setIdEmailVoluntario(Long idEmailVoluntario) {
        this.idEmailVoluntario = idEmailVoluntario;
    }

        return email;
    }

package com.pi.demo.entities;

import jakarta.persistence.*;

@Entity
@Table(name = "tb_voluntarios")
public class Voluntario {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    @Column(name = "idVoluntario")
    private Long idVoluntario;

    @Column(name = "nome", nullable = false, length = 45)
    private String nome;

    @Column(name = "idade", nullable = false)
    private Integer idade;

    @OneToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "tb_emailVoluntario_idEmailVoluntario", nullable = false)
    private EmailVoluntario emailVoluntario;

    public Voluntario() {}

    public Voluntario(Long idVoluntario, String nome, Integer idade, EmailVoluntario emailVoluntario) {
        this.idVoluntario = idVoluntario;
        this.nome = nome;
        this.idade = idade;
        this.emailVoluntario = emailVoluntario;
    }

    public Long getIdVoluntario() {
        return idVoluntario;
    }

    public void setIdVoluntario(Long idVoluntario) {
        this.idVoluntario = idVoluntario;
    }

    public String getNome() {
        return nome;
    }

    public void setNome(String nome) {
        this.nome = nome;
    }

    public Integer getIdade() {
        return idade;
    }

    public void setIdade(Integer idade) {
        this.idade = idade;
    }

    public EmailVoluntario getEmailVoluntario() {
        return emailVoluntario;
    }

    public void setEmailVoluntario(EmailVoluntario emailVoluntario) {
        this.emailVoluntario = emailVoluntario;
    }
}

package com.pi.demo.controllers;

import com.pi.demo.entities.EmailVoluntario;
import com.pi.demo.services.EmailVoluntarioService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/api/v1/emailVoluntarios")
public class EmailVoluntarioController {

    @Autowired
    private EmailVoluntarioService emailVoluntarioService;

    @GetMapping
    public ResponseEntity<List<EmailVoluntario>> getAllEmailVoluntarios() {
        return ResponseEntity.ok(emailVoluntarioService.findAll());
    }

    @GetMapping("/{id}")
    public ResponseEntity<EmailVoluntario> getEmailVoluntarioById(@PathVariable Long id) {
        return emailVoluntarioService.findById(id)
                .map(ResponseEntity::ok)
                .orElse(ResponseEntity.notFound().build());
    }

    @PostMapping
    public ResponseEntity<EmailVoluntario> createEmailVoluntario(@RequestBody EmailVoluntario emailVoluntario) {
        return ResponseEntity.ok(emailVoluntarioService.save(emailVoluntario));
    }

    @PutMapping("/{id}")
    public ResponseEntity<EmailVoluntario> updateEmailVoluntario(@PathVariable Long id, @RequestBody EmailVoluntario emailVoluntario) {
        if (!emailVoluntarioService.findById(id).isPresent()) {
            return ResponseEntity.notFound().build();
        }
        emailVoluntario.setIdEmailVoluntario(id);
        return ResponseEntity.ok(emailVoluntarioService.save(emailVoluntario));
    }

    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deleteEmailVoluntario(@PathVariable Long id) {
        if (!emailVoluntarioService.findById(id).isPresent()) {
            return ResponseEntity.notFound().build();
        }
        emailVoluntarioService.deleteById(id);
        return ResponseEntity.noContent().build();
    }
}
