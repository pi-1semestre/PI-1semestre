# 🐾 Projeto: Apoio Pet

Bem-vindo(a) à página do **Apoio Pet**! Este projeto nasceu com um propósito especial: ajudar a conectar animais que precisam de um novo lar com pessoas dispostas a adotar, tornando o trabalho da nossa ONG muito mais organizado e eficiente.

## 🌟 O que é o Apoio Pet?

O Apoio Pet é uma plataforma digital (site) que funciona como um "ponto de encontro" para a adoção responsável. Sabemos que organizar todas as informações sobre os animais e gerenciar pedidos de adoção pode ser um desafio para as ONGs. Por isso, criamos um espaço onde tudo é centralizado, fácil de usar e muito visual.

## 🎯 Nossos Objetivos

Queremos tornar o processo de adoção uma experiência positiva para todos:

* **Para quem quer adotar:** Facilitar a busca pelo seu futuro melhor amigo, com fotos e informações claras.
* **Para a ONG:** Oferecer ferramentas simples para gerenciar os animais (quem está disponível, quem já foi adotado) e receber pedidos de interessados.
* **Para a comunidade:** Criar um canal de comunicação para espalhar a importância da posse responsável e compartilhar histórias de superação dos nossos pets.

## ✨ O que você encontra no site?

* **Catálogo de Pets:** Você pode ver fotos e saber tudo sobre os animais disponíveis, filtrando por espécie, tamanho ou idade.
* **Histórias de Sucesso:** Um espaço para ver relatos emocionantes de animais que já encontraram um lar.
* **Como Ajudar:** Informações práticas sobre como fazer doações para manter o trabalho da ONG funcionando.
* **Denúncias e Resgates:** Um caminho claro para quem precisa comunicar uma situação de risco envolvendo animais.
